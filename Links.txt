File links for all references in wiki,
for now mostly pictures and css-stylings. (30.06.21)

To upload files in general, write the following in the box "Destination filename":
T--NTNU-Trondheim--
before the filename you'd like, and then the file type you're uploading.

To reference templates (the css files you upload in a very nasty manner)
for your pages, use the following method:
1. Main shell: {{Template:NTNU-Trondheim
            +
2. Specific CSS-file /File/Directory}}
